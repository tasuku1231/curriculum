IT基本用語
■ IT
　・IT（Information Technology）
    コンピューターとネットワークを利用した技術の総称

　・ICT（Information and Communication Technology）
    ITとほとんど同じ。
    違いとしてはSNSが発展しコミュニケーションが重要視されるようになりI"C"Tとなった。

　・loT（Internet of Things）
    身の回りのあらゆるモノがインターネットにつながる仕組み。（電子機器がインターネットに繋がる）

■ OS
　・OS（オペレーティングシステム）
    パソコンの心臓部分。OSがなければただの空の箱と同じ。
    OSの仕事は端末全体を管理、制御すること。

　・Windows
    Microsoftが提供しているOS

　・macOS (Mac OS X)
    アップルが提供しているOS。
    Mac OS Xは他の端末が”OS”に対しOS Xのみ後ろに名前がついていたので改名した。

　・UNIX
    現存する最も古いOS。
    特徴は安全性、機能が少ないので軽い、安い。カーナビや携帯電話などに使われていた。

　・Linux
    UNIXととても似ているOS。UNIXを元にLinuxが開発された。
    スマートフォンなどに使われている。

■ 言語・環境
　・マークアップ／フロントエンド
    マークアップは主にHTLMでの作成、実装を担当。フロントエンドはHTLMの他にCSS,JavaScriptの知識経験が豊富。
    マークアップ＜フロントエンドみたいな感じ。

　・HTLM
    ホームページなどのWEBサイトを作成する際に使われるプログラミング言語。

　・CSS
    HTLMと合わせて使われる言語。
    HTMLは文章を入力し、CSSはその文章をデザインする。

　・JavaScript
    CSSでは静止的なのに対してJavaScriptは動的な動き（アニメーション）をする事ができる。

　・サーバーサイド
    PCやスマホをインターネット越しにサーバにアクセスする事でデータやサービスを提供されている。（ツイッター、インスタ）
    つまり大元の事。

　・.NET（C#, VB）
    Windowsのシステムやアプリケーション、開発環境の用語。
    C#, VBの言語に使われる。

　・Java
    プログラミング言語の一つ。汎用性が高くOSに依存しない為、、世界中で多く使用されている。

　・Java SE
    Javaの基本機能をまとめたもの。PCで動作するゲームや便利なソフトを作ることが出来る。

　・Java EE（Jakarta EE）
    JavaSEを元にしてサーバーサイドの機能を追加したもの。（拡張機能）
    拡張機能を使うことでJavaでのWebアプリ、サイトを作ることができる。

　・Perl（パール）
    CGI(拡張機能みないなもの)を実現するためのプログラミング言語の1つ
    テキスト処理が得意。

　・PHP
    Webページを生成することができるサーバーサイドのスクリプト言語（コーディングが比較的に簡単な言語）
    HTMLに組み込むことで動的な表現が可能。

　・Ruby
    スクリプト言語。Webアプリケーションで多く使用されている。（クックパッドなど）
    メリットとして一番大きいのは、Ruby on Railsという
    Webアプリケーションフレームワーク（Webアプリやシステムを開発するのに必要な機能が予め用意された骨組み）を使うことができる。

■ サーバーソフト
　・Apache（アパッチ）
    世界中で使用されているWebサーバーソフトウェア。サーバーを構築するにはWebサーバーソフトウェアが必須。
    安心安全安定

　・nginx（エンジンエックス）
    Apacheの次に人気。処理速度がはやい。

　・IIS
    WindowsのWebサーバーソフトウェア。
    Windows用途で作られている為、とても使いやすい。

■ SQL
　・SQL
    データベース言語でありデータベースを管理するソフトウェアを操作、制御をすることが目的。
    プログラミング言語ではないのでシステムは開発できない、ユーザーが指定したモノを見つけ出すためのもの。

　　・DB2
    データベース管理システムの一つ。SQLを初めて採用した関係データベース管理システムらしい

　　・MySQL
    データベース管理システムの一つ。大容量のデータを高速に行える。レンタルサーバーや検索エンジンに使用されている。
    MariaDBはMySQLから派生したもので高性能。

　　・Oracle DB（オラクル　データベース）
    米国オラクル が開発・販売している、関係データベース管理システム
    高い堅守性。

　　・PostgreSQL（ポストグレスキューエル）
    オープンソフトウエア(無料)なのでコストが安く、様々な様々なシステムに対応している。
    商用のデータベースと遜色ない性能。

　　・SQLite
    無料のデータベース。サーバーとしてではなくアプリに組み込む。
    軽量、コンパクト、省メモリ

　　・SQL Server
    Microsoftが開発したデータベース管理システム。MS SQLともいう。
    SQL Serverではデータベースを視覚的に操作する「GUI」が利用できる。

　・CRUD（クラッド）
    ほとんど全てのコンピューターソフトウェア永続性の４つの基本機能のイニシャルを並べた用語。
    生成、読み取り、更新、削除

　・トランザクション
    コンピュータシステムにおける、永続的なデータに対する不可分な一連の処理
　　・コミット
    確定させる。反映させる。

　　・ロールバック
    データ障害が発生した際、発生する前の状態まで戻る事。

　・排他制御（排他ロック）
    データ同時更新を防ぐ為のもの
    ・楽観ロック
    めったなことでは他者との同時更新は起きないであろう、という楽観的な前提の排他制御
    ・悲観ロック
    他者が同じデータに頻繁に変更を加えるであろう、という悲観的な前提の排他制御
